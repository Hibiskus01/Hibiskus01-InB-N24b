JavaScript Probleme

Verbessere deine JavaScript-Fähigkeiten mit diesen anfängerfreundlichen Herausforderungen.

Problem 1: "Hallo, Welt!" ausgeben

Schreibe eine Funktion, die "Hallo, Welt!" in die Konsole ausgibt.
Problem 2: Zwei Zahlen addieren

Erstelle eine Funktion, die zwei Zahlen als Eingabe nimmt und ihre Summe zurückgibt.
Problem 3: Prüfen, ob eine Zahl gerade ist

Schreibe eine Funktion, die überprüft, ob eine gegebene Zahl gerade oder ungerade ist.
Problem 4: Das Quadrat einer Zahl berechnen

Schreibe eine Funktion, die eine Zahl nimmt und ihr Quadrat zurückgibt.
Problem 5: Hallo, Welt! 2.0

Erstelle eine Funktion, die "Hallo, Welt!" in die Konsole ausgibt.
Problem 6: Die Länge eines Strings ermitteln

Schreibe eine Funktion, die einen String nimmt und seine Länge zurückgibt.
Problem 7: Das erste Element eines Arrays zurückgeben

Erstelle eine Funktion, die das erste Element eines Arrays zurückgibt.
Problem 8: Prüfen, ob eine Zahl positiv ist

Schreibe eine Funktion, die überprüft, ob eine gegebene Zahl positiv, negativ oder null ist.
Problem 9: Einen String N-mal wiederholen

Erstelle eine Funktion, die einen String eine bestimmte Anzahl von Malen wiederholt.
Problem 10: Den Rest einer Division berechnen

Schreibe eine Funktion, die zwei Zahlen nimmt und den Rest ihrer Division zurückgibt.
